# Polymarket Analytics Skill

Polymarket 预测市场智能分析工具包 - **自动进化版**

## 🌟 核心亮点

### 自动进化系统

区别于传统静态分析工具，本系统能够：

1. **自动学习** - 扫描钱包时自动识别交易模式
2. **原则生成** - 基于成功模式自动生成交易原则
3. **信号验证** - 追踪并验证每个原则的准确度
4. **持续进化** - 淘汰低效原则，优化高准确度原则
5. **智能建议** - 基于进化后的原则库提供实时建议

## 功能模块

| 模块 | 脚本 | 功能描述 | 自动进化 |
|------|------|----------|----------|
| **智能扫描** | `smart_scanner.py` | 集成学习与进化的钱包扫描 | ✅ |
| **自动进化** | `auto_evolution.py` | 核心进化引擎 | ✅ |
| **钱包分析** | `wallet_analysis.py` | 基础钱包分析 | ❌ |
| **模拟回测** | `backtesting.py` | 模拟跟随回测 | ❌ |
| **市场分析** | `market_analysis.py` | 市场热点分析 | ❌ |
| **策略建议** | `strategy_suggestions.py` | 生成交易建议 | ❌ |
| **原则进化** | `self_evolution.py` | 策略参数优化 | ⚠️ 手动 |

```bash

  -b 0xaddr1 0xaddr2 0xaddr3 0xaddr4 0xaddr5

# - 分析每个钱包
# - 学习交易模式
```

```bash

python3 scripts/smart_scanner.py \
  -m 0xsmart1 0xsmart2 0xsmart3 \
  --interval 30
```

## 交易原则库

系统维护一个动态原则库，示例：

```json
{
  "id": "p_wallet123_win",
  "name": "高胜率钱包跟随策略",
  "condition": "wallet.win_rate > 60 AND wallet.pnl > 5000",
  "action": "跟随买入，仓位5%，止盈30%止损15%",
  "confidence": 75.0,
  "success_count": 15,
  "fail_count": 5,
  "accuracy": 75.0,
  "created_at": "2026-03-04T10:00:00",
  "last_updated": "2026-03-04T16:00:00"
}
```

**原则状态**：
- 🟢 **高准确度** (>60%) - 信心度提升，优先推荐
- 🟡 **中等准确度** (40-60%) - 保留观察，谨慎使用
- 🔴 **低准确度** (<40%) - 逐渐淘汰
- ⚪ **未验证** - 新生成，等待数据


## 高级用法

### 手动触发进化

```bash
# 验证所有历史信号
python3 scripts/auto_evolution.py -v

# 触发原则进化
python3 scripts/auto_evolution.py -e

# 查看进化统计
python3 scripts/auto_evolution.py -l
```

### 自定义扫描

```bash
# 只分析不学习
python3 scripts/smart_scanner.py -w 0xaddr --no-learn

# 显示当前原则库
python3 scripts/smart_scanner.py --principles
```

## 风险提示

⚠️ **重要声明**:
- 本工具仅用于数据分析，不构成投资建议
- 预测市场交易存在高风险，可能损失全部本金
- 历史表现不代表未来收益
- 自动进化系统需要足够数据（建议扫描20+钱包）才能达到稳定准确度
- 请根据自身风险承受能力谨慎决策
